INSTALLATION RAPIDE

1. Ouvre ton index.html
2. Copie le contenu de :
   - live-chat.html

3. Ouvre ton style.css
4. Copie le contenu de :
   - live-chat.css

5. Ouvre ton script.js
6. Colle le contenu de :
   - live-chat.js

7. Dans initFirebase() ajoute :
   initChat();

8. Dans Firebase > Firestore Rules :
   colle le contenu de firestore-rules.txt
