TEAM RDM - SITE TWITCH

1) Extraire le ZIP
2) Double-cliquer sur start.bat
3) Ouvrir http://localhost:8000

NOUVEAUTÉ - PUBLICATION DE CLIPS PAR LES MEMBRES:
- Va dans la section CLIPS puis "Publier un clip Twitch".
- Entre ton pseudo Twitch, un titre, le lien du clip Twitch et le code membre.
- Code membre par défaut: RDM5996
- Membres autorisés: kenshin5996, c_djo, manimang0, fandeipromxtrollmod, theoherlintw, maszoks.
- Les clips ajoutés sont enregistrés dans le navigateur avec localStorage.
- Important: sur un site statique, les clips ne sont pas partagés automatiquement avec tous les visiteurs. Pour rendre les publications visibles à toute la team, il faut ajouter un backend / une base de données.

Le site récupère les photos Twitch avec un service public d'avatar.
Si une photo ne se charge pas, le logo loup RDM s'affiche automatiquement.
Si Twitch refuse le lecteur blanc, clique sur OUVRIR TWITCH : cela vient du blocage Twitch/navigateur ou bloqueur de pub.
